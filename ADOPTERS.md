# Adopters

Organizations and teams that have adopted Constitutional Governance.

---

*Be the first to add your organization. Open a pull request.*

---

## How to add your organization

1. Fork this repository
2. Add an entry to this file in the format below
3. Open a pull request

```markdown
### [Organization name](https://link-to-org-or-case-study)

**Domain(s) governed:** Kafka platform, REST APIs, ...
**Implementation:** [Nomos](implementations/nomos.md) / custom
**Since:** YYYY
**Notes:** Optional short description
```

---

*Constitutional Governance is an open methodology. Adopters are not required to use any specific tool or to open-source their governance repository. An entry here means only that the organization has adopted the methodology and wishes to be recognized.*
